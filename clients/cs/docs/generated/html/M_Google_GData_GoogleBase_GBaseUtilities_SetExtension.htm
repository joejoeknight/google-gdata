GBaseUtilities..::.SetExtension Method
GBaseUtilities Class  See Also  Send Feedback
Replaces or adds an extension of the given type to the list. This method looks for the first element on the list of the given type and replaces it with the new value. If there are no element of this type yet, a new element is added at the end of the list.
