The kind parameter lets you request information about a particular kind of item. The parameter value should be a comma-separated list of requested kinds. If you omit the kind parameter, Picasa Web Albums chooses a default kind depending on the level of feed you're requesting. For a user-based feed, the default kind is album; for an album-based feed, the default kind is photo; for a photo-based feed, the default kind is comment; for a community search feed, the default kind is photo.
